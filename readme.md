Installation:
------------------------------------
1: Extract the downloaded .zip file in a convenient location

2: upload the '/theme/twitterize/' folder to '/wp-content/themes/' directory in your web server.

3: Log in to the WordPress Administration Dashboard

4: Select the 'Themes' sub-menu from 'Appearance' menu

5: Select the 'Twitterize' theme from the available list of themes to preview

6: Activate the theme.
